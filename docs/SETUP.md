# Kurulum

1. n8n kurulu bir ortama sahip olun.
2. Projede `.env.example` dosyasını `.env` olarak kopyalayın.
3. API anahtarlarını `.env` içine girin.
4. n8n arayüzünde `workflows/` altındaki JSON dosyalarını import edin.
5. Webhook node path ve endpoint alanlarını kendi ortam URL'nize göre düzenleyin.
