# n8n-project

AI destekli doküman işleme ve sohbet akışları için minimal n8n proje iskeleti.

## Hızlı Başlangıç

1. `.env.example` dosyasını `.env` olarak kopyalayın.
2. Gerekli API anahtarlarını doldurun.
3. `workflows/` içindeki JSON dosyalarını n8n'e import edin.
4. Webhook URL'lerini n8n ortamınıza göre güncelleyin.

## Klasörler

- `workflows/`: 7 temel n8n akış şablonu
- `config/`: temel yapılandırma ve API key template
- `docs/`: kurulum, akış ve API entegrasyon notları
